# Fabulously Optimized

A Minecraft modpack shared with the [Brassworks Launcher](https://github.com/brassworks-smp).

## Install

Open this pack in the Brassworks Launcher to play it (and get automatic updates):

`https://raw.githubusercontent.com/diamondino/fabulously-optimized/main/pack.toml`

This repository is published automatically by the launcher — edits made here may be overwritten on the next update.
